jQuery Bing maps extend with pagination

Using jquery with Bing maps

Download jQuery 1.4.X or higher or use Googles or Microsofts CDN.

<script type="text/javascript" src="http://ecn.dev.virtualearth.net/mapcontrol/mapcontrol.ashx?v=7.0"></script>
<script type="text/javascript" src="http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.7.min.js"></script>
<script type="text/javascript" src="PATH_TO_PLUGIN/jquery.fn.map.js"></script>

An example of how to extend the plugin with a simple pagination (see jquery.ui.bmap.extensions.js).

After extending the plugin, just add some markers and call $('#map_canvas').gmap('pagination');

$('#map_canvas').gmap({'credentials': YOUR_BING_MAP_CREDENTIALS, 'zoom':5}).bind('init', function(ev, map) {
	$('#map_canvas').gmap('addMarker', 
		{
			'location': '57.7973333,12.0502107', 
			'title': 'Something to show'
		}
	).click(function() {
		$('#map_canvas').gmap('openInfoWindow', {'title': this.target.title}, this);
	});
	$('#map_canvas').gmap('pagination');
});
